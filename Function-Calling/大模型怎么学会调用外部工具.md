# 大模型怎么学会调用外部工具？

## 题目

做了这么多 Agent 项目，大模型怎么学会调用外部工具都说不清？

## 核心知识点

**Tool Use 的训练方式 / Function Calling 原理**

## 答案要点

1. **Function Calling 的训练方式**：
   - 在预训练阶段：模型学习大量包含「工具调用示例」的文本
   - 在微调阶段：专门用「工具调用」类型的数据集进行训练
   - 典型数据集格式：`用户问题 → {tool_name, arguments} → 工具返回 → 最终回答`

2. **让 LLM 学会调用工具的关键**：
   - **工具描述（Tool Description）**：告诉 LLM 有什么工具、参数是什么
   - **示例训练**：让模型看到大量「什么时候该调什么工具」的样例
   - **输出格式约束**：让模型学会输出结构化的调用指令

3. **Function Calling vs Tool Use**：
   - Function Calling 是 OpenAI 提出的概念，是 Tool Use 的一种实现
   - Tool Use 是更广泛的概念，包括各种让 LLM 操作工具的方法
