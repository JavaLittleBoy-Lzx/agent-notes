# 大模型怎么学会调用外部工具？

> [!abstract] 一句话回答
> 在**预训练阶段**学习大量工具调用示例 + 在**微调阶段**用专门的工具调用数据集训练，让模型理解"什么时候该调什么工具、怎么调"。

---

## 🧠 训练方式

### 1. 预训练阶段：泛化能力

- 模型在海量文本中学习语言理解能力
- 其中包含大量"人类描述工具使用"的语料
- 学会：工具是什么、怎么描述一个工具

### 2. 微调阶段：专精任务

用专门的**Function Calling 数据集**微调，格式：

```
输入：用户问题（如"北京天气怎么样？"）
期望输出：
  {
    "tool_call": {
      "name": "get_weather",
      "arguments": {"city": "北京"}
    }
  }
```

---

## 🔑 三个关键要素

| 要素 | 作用 |
|------|------|
| **工具描述（Tool Description）** | 让模型知道有哪些工具可用、参数是什么 |
| **示例训练（Few-shot Examples）** | 让模型学会"什么场景调什么工具" |
| **输出格式约束** | 让模型输出结构化的调用指令，而不是自然语言 |

---

## 🎯 面试考察点

1. **训练方式** — 预训练 + 微调
2. **Function Calling 本质** — 不是 LLM 自己会调用工具，而是训练出来的"调用习惯"
3. **三个关键要素** — 工具描述 / 示例 / 输出格式
