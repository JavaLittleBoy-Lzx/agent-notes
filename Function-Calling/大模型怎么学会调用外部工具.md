## 🔑 三个关键要素

| 要素 | 作用 |
|------|------|
| **工具描述（Tool Description）** | 让模型知道有哪些工具可用、参数是什么 |
| **示例训练（Few-shot Examples）** | 让模型学会"什么场景调什么工具" |
| **输出格式约束** | 让模型输出结构化的调用指令，而不是自然语言 |

---

## 🎯 面试考察点

1. **训练方式** — 预训练 + 微调
2. **Function Calling 本质** — 不是 LLM 自己会调用工具，而是训练出来的"调用习惯"
3. **三个关键要素** — 工具描述 / 示例 / 输出格式
