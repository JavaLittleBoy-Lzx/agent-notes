# 大模型是怎么查天气的？

## 题目

大模型是怎么查天气的？它自己去调 API？

## 核心知识点

**Function Calling / Tool Use 机制**

## 答案要点

1. **LLM 本身是纯语言处理器**，不能直接上网、发请求、执行代码
2. 通过 **Function Calling（函数调用）** 机制，LLM 可以：
   - 理解用户意图：「查天气」需要调用外部天气 API
   - 识别需要调用的工具：`get_weather()`
   - 生成结构化的调用参数：`{"location": "北京"}`
3. **程序负责真正执行**：你的代码接收 LLM 返回的调用指令，实际调用天气 API
4. 将 API 返回结果反馈给 LLM，LLM 再生成自然语言回答

## 关键分工

- **LLM 负责「决定做什么」** → 返回 `{tool_name, arguments}`
- **程序负责「真正执行」** → 调用 API，返回结果给 LLM
