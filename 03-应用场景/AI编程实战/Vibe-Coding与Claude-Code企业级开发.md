## 核心内容

### Vibe Coding 时代的到来

程序员的核心任务正在从**「写代码」**转向**「做决策」**。

从 Copilot、Cursor 到 Claude Code，AI 编程工具正以惊人速度迭代。

### 案例：一人一周，用 Claude Code 从零构建 Hify

Hify 是一个对标 Dify 的本地 AI Agent 开发平台，支持：
- 多模型管理
- Agent 配置
- 对话引擎
- RAG 知识库
- 工作流编排
- MCP 工具接入

> [!tip] 关键数据
> Robert（一线大厂基础设施工程师）用 Claude Code，**一人，一周**，从零完成。

### 真正提效的瓶颈在哪里？

> [!warning] 核心洞察
> 为什么大多数程序员用 AI 工具提效有限？真正的瓶颈**从来不在 AI 的能力上**。

### 课程：《Claude Code 企业级全链路开发实战》

主要内容：
1. **认知转变**：从「我写代码」到「我指挥 AI 写代码」
2. **规范驱动开发（SDD）**：让 AI 输出符合预期
3. **顶层设计**：先架构决策，再一行行代码落地
4. **全程实操**：后端骨架、前端工程、核心功能交付
5. **容器化部署**：从本地可跑到生产可交付
