# 为什么需要 Agent

> 来源：微信公众号 - 小林coding
> 美团面试题

## 面试题

美团面试官：「既然大模型已经这么强了，为什么还要做 Agent？」

---

## 核心回答

大模型虽强，但有根本局限：

| 大模型的局限 | Agent 的解决方案 |
|---|---|
| 知识有时效，无法实时获取 | 调用搜索/数据库工具 |
| 无法执行操作（发邮件、发消息） | 集成执行工具 |
| 只能处理单次请求，无状态 | 记忆系统保持上下文 |
| 无法处理复杂多步任务 | 规划模块拆解任务 |

**一句话总结：Agent = LLM + 工具 + 记忆 + 规划，弥补纯 LLM 的短板。**
