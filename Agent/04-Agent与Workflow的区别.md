# Agent 与 Workflow 的区别

## 面试题

腾讯三面面试官：「Agent 和 Workflow 的区别是什么？」

---

## 核心要点

**Agent = 自主决策 + 动态执行**
**Workflow = 预设流程 + 固定路径**

| | Agent | Workflow |
|---|---|---|
| 决策方式 | LLM 自主判断下一步做什么 | 预设流程，节点固定 |
| 执行路径 | 动态，根据每步结果调整 | 固定流程图 |
| 灵活性 | 高，可应对未知场景 | 低，适合固定业务流程 |
| 适用场景 | 复杂推理、多工具调用 | 固定审批流、数据处理流水线 |
