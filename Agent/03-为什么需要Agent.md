## 🔧 为什么 Agent 能解决？

```
┌─────────────────────────────────┐
│           Agent 系统             │
│                                  │
│  ┌──────────┐                    │
│  │   LLM   │ ← 大脑，理解+决策   │
│  └────┬────┘                    │
│       │                          │
│  ┌────▼────┐                    │
│  │  工具   │ ← 联网、查数据库... │
│  └────┬────┘                    │
│       │                          │
│  ┌────▼────┐                    │
│  │  记忆   │ ← 跨对话保持状态    │
│  └────┬────┘                    │
│       │                          │
│  ┌────▼────┐                    │
│  │  规划   │ ← 拆解复杂任务     │
│  └─────────┘                    │
└─────────────────────────────────┘
```

---

## ⚖️ 对比：纯 LLM vs Agent

| 能力 | 纯 LLM | Agent |
|------|---------|-------|
| 回答通用问题 | ✅ | ✅ |
| 实时查股价/天气 | ❌ | ✅ |
| 发邮件通知 | ❌ | ✅ |
| 跨多步保持上下文 | ❌ | ✅ |
| 处理复杂多步任务 | 弱 | ✅ |

---

## 💡 一句话总结

> **AI 是放大器，Agent 是执行器。**
> - LLM 给你答案
> - Agent 帮你做事

---

## 🎯 面试考察点

1. **大模型的三个局限** — 知识时效/无法执行/无状态
2. **Agent 如何弥补** — 工具+记忆+规划
3. **Agent 的核心价值** — 从"回答问题"到"完成任务"
