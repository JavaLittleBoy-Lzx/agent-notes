# Vibe Coding 与 Claude Code 实战

> [!abstract] 摘要
> AI 编程工具正从「辅助」进入 Vibe Coding 时代。本文介绍如何用 Claude Code 从零构建企业级 AI Agent 平台（Hify），核心观点：AI 是放大器不是发动机，有方向才能放大。

---

## 核心观点

> [!quote] 核心观点
> AI 是放大器，不是发动机。你有方向它放大 10 倍，你没方向它放大的是 0。

- **Vibe Coding 时代**：程序员核心任务从「写代码」转向「做决策」

---

## 案例：用 Claude Code 一周构建简版 Dify

**目标**：从零搭建支持多模型管理、Agent 配置、对话交互、MCP 工具接入的本地 AI Agent 平台（Hify）

**所需时间**：一线大厂工程师 Robert 的答案是 **一周**

**涉及技术**：
- 模型管理
- Agent 配置
- 对话引擎
- RAG 知识库
- 工作流编排
- MCP 工具接入

## 课程大纲：《Claude Code 企业级全链路开发实战》

### 一、认知对齐：架构师 + AI 团队

- 从「我写代码」到「我指挥 AI 写代码」
